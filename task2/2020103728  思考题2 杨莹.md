##1、读取数据，每行数据读取为一个元组，并将元组存储在列表中
```
res = []
with open('part.tbl', 'r') as f:
    line = ' '
    while line:
        line = f.readline()
        if len(line)>0:
            data = line.strip().split('|')
            res.append(tuple(data))
```
###该列表共有20000行
